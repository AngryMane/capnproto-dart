# capnpc_dart

The Dart code-generator plugin for the official `capnp` compiler.

```sh
dart pub global activate capnpc_dart
capnp compile -o dart:lib/src/generated schema.capnp
```

See the [schema and code generation guide](https://angrymane.github.io/capnproto-dart/howto/schema-and-codegen)
for imports and compatibility checking.
