# capnproto_dart_rpc

Pure Dart Cap'n Proto RPC support for two-party Level 1 connections over TCP
or WebSocket. The package supports callbacks, promise pipelining, tail calls,
and Resolve/Disembargo in both directions.

See the [RPC guide](https://angrymane.github.io/capnproto-dart/howto/rpc) and
the [support matrix](https://github.com/AngryMane/capnproto-dart#rpc-support-status)
for usage and current protocol boundaries.
