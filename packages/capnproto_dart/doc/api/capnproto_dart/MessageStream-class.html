MessageStream

MessageStream class

Utilities for reading and writing streams of Cap'n Proto messages.

Each message in a stream is framed identically to the single-message format produced by MessageBuilder.serialize / read by MessageReader.deserialize: a little-endian header followed by segment data. Messages are concatenated back-to-back with no additional delimiters.

Intended users

  • Application and library developers working with Cap'n Proto messages.

Primary use cases

  • Reads, writes, validates, or reports failures for application messages.
