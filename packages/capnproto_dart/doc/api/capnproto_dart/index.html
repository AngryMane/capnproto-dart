Typedefs

SchemaRegistry = Map<int, SchemaInfo>
Maps a schema node id to its SchemaInfo (struct or enum), so encodeText/decodeText can resolve the nested struct/enum types referenced by field types. Node ids are globally unique, so one registry can cover an entire schema file (or several).

Exceptions / Errors

CapnpException
Reports a Cap'n Proto failure with a category and optional cause.
DecodeException
Reports malformed input or a configured decoding-limit violation.
