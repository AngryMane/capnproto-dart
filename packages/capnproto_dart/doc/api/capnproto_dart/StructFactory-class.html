StructFactory

StructFactory<R extends StructReader, B extends StructBuilder> class abstract

Factory that carries struct layout information and constructs typed reader/builder instances from raw arena pointers.

Generated code produces a concrete subclass for each Cap'n Proto struct.

Intended users

  • Authors of generated capnpc_dart bindings and low-level runtime integrations.

Primary use cases

  • Supports the typed bindings that map schema declarations to the Cap'n Proto wire layout.
