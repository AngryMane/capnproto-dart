StructBuilder

StructBuilder class abstract

Base class for all generated struct builders.

Subclasses expose typed setters and init methods by calling the helpers below. The helpers apply XOR-based default masking per the Cap'n Proto spec.

Intended users

  • Authors of generated capnpc_dart bindings and low-level runtime integrations.

Primary use cases

  • Supports the typed bindings that map schema declarations to the Cap'n Proto wire layout.
Implementers
  • DynamicStructBuilder

adoptPointerField(int ptrIndex, Orphan? orphan) → void
Adopts orphan into the pointer at ptrIndex (zero-copy — see orphan.dart), replacing whatever was there. Passing null clears the field. Object content is never copied, although cross-segment adoption allocates a two-word double-far landing pad. On failure, orphan remains available for another adoption attempt.

Detaches the pointer at ptrIndex as an Orphan (zero-copy — see orphan.dart), leaving the field unset. Returns null if the field was already unset. Existing builders and readers are not invalidated; they remain live aliases of the detached object's arena memory.
