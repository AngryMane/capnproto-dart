RawStructReader

RawStructReader class

Untyped view of a struct's memory inside a segment, used by generated readers.

The data section occupies words [dataWordOffset, dataWordOffset + dataWords). The pointer section occupies words [ptrWordOffset, ptrWordOffset + ptrWords).

Intended users

  • Authors of generated capnpc_dart bindings and low-level runtime integrations.

Primary use cases

  • Supports the typed bindings that map schema declarations to the Cap'n Proto wire layout.
