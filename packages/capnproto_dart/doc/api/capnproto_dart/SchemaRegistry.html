SchemaRegistry typedef

SchemaRegistry = Map<int, SchemaInfo>

Maps a schema node id to its SchemaInfo (struct or enum), so encodeText/decodeText can resolve the nested struct/enum types referenced by field types. Node ids are globally unique, so one registry can cover an entire schema file (or several).

**Intended users**
* Application and tooling developers using the public runtime API.

**Primary use cases**
* Converts messages or schema metadata for serialization, diagnostics, or dynamic processing.

**Parameters**
* Function parameters supply the source value and any schema, options, or conversion callbacks required by the operation.

**Returns**
* The converted message, text, or schema-registry value.
