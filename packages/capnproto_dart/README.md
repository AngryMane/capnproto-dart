# capnproto_dart

Pure Dart Cap'n Proto serialization, packed encoding, canonicalization, and
streaming support. It has no FFI dependency.

See the [repository README](https://github.com/AngryMane/capnproto-dart#quick-start)
for setup and generated-code examples, and the
[serialization guide](https://angrymane.github.io/capnproto-dart/howto/serialization)
for the complete API overview.
