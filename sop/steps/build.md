* Coding
  * 必須: true
  * 種別: 指示
  * 内容: Designで決定した内容をAIが実装する
  * 指示: ./docs/internal-design.md と ./docs/boundary-design.md に従い、./docs/scope.md で定義した機能を実装してください。1機能ずつ段階的に実装し、各機能の実装完了後に報告してください。
  * 出力: 定義しない
  * 完了条件: Scope で定義した全機能が実装され、コードがビルドに成功し、ユーザーが動作をレビューして承認した。

  * 種別: 指示
  * 内容: Codingに対するテストを実装する
  * 指示: 実装した各機能に対してテストを実装してください。単体テストと結合テストを含め、正常系・異常系・境界値を網羅するテストケースを作成してください。
  * 出力: 定義しない
  * 完了条件: 全テストが成功し、ユーザーがテストケースをレビューして承認した。
