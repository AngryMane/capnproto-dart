* Consistency Check
  * 必須: false
  * 種別: 指示
  * 内容: ドキュメントと実装の整合性を確認する
  * 指示: 以下の観点で整合性を確認し、矛盾・乖離があれば報告してください。1. docs間の矛盾（usecase / scope / design / README が互いに矛盾していないか）2. scope と実装の差分（scope.md で定義した機能が実装されているか）3. design と実装の差分（boundary-design.md / internal-design.md と実装が一致しているか）4. README と実装の差分（README.md の使い方・APIが実装と一致しているか）
  * 出力: 定義しない
  * 完了条件: 整合性確認が完了し、矛盾・乖離の有無をユーザーに報告した。
